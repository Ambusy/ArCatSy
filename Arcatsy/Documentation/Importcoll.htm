Import Collection data or definitions


On the main menu

click Maintenance(), and on the maintenance screen
select the collection to import () and click Import collection().

Import Collection definitions

To create a new collection, almost identical to an existing one, you may export the definition of a collection to a file, edit the text if you wish, and then import these definitions into a new collection. Arcatsy then allows to adjust it further.

On the screen that appears,

click Browse () to choose the name of the file that contains the definitions.
check Definitions ().
And finally click Import()

The collection will be imported and you'll return to the maintenance screen.


Import Collection data

To include data from outsize ArCatSy or to copy data from one pc to another, you may import data.

On the screen that appears,

click Browse () to choose the name of the file to receive the data.
check Data ().
Optionally choose the correct separator () if different from the TAB-symbol.
And finally click Import()

The collection will be imported. If the internal sequencenumber (with variablename #) is present in the file, this number will be used to identify the entries, if absent the entries will be added as usual at the end of the collection.
Finally you'll return to the maintenance screen.
