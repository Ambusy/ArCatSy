Display entities

To display collection data, on the main menu

click on Print () if you previously saved a keylist in a search. But you will get here automatically after clicking the button on the search, the sort or the register window. The default layouts will be used to display the contents of the documents to be shown. Click on the “Use a different layout” to interrupt and to choose any non-default layout.

A summary screen

then shows each collection, with the number of entities found between brackets, and a list of layouts for each collection. All collections and the default layouts are cheched, but you may uncheck a collection or choose a different layout.
Click Select a keylist () if you want to switch to another result list, saved previously in a search, a sort or a register. You'll be asked to identify the list by it's name.

Click Show results () to view the entities.

Click Print the results () to print the list as displayed on the screen
Click Use a different layout () to change layout(s) You'll return to the initial screen with the checkboxes.
Click View HTML () to view the HTML, in case of difficulty to get a layout right. You'll see the intermediate result file and may not proceed until you close it.

You may write a ReXX program to edit the text of a field, just before it is displayed. See Display and print for details.

You may write your own HTML header lines, using system messages 204 to 207. See the "system messages.txt" file for details.
