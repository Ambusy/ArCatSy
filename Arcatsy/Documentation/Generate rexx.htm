Generate a ReXX Data Entry program

To insert or modify data of an entity in a collection, you need te have a ReXX program, that guides the data entry process.
ArCatSy provides for generation of an acceptable program. This program may be adapted by you, to add flexibility to the process.

On the main menu, choose for Maintenance to get here:

Select the collection, for which you want to generate the program () and click on Generate ()
You'll get a confirmation screen

where you click on OK (), to generate the program with the name as displayed.

It is up to you, if you wish, to modify the generated program. The generated version is fully operational, it askes for all fields and does no editing or other tricks.

To modify a data entry program, see the guidelines in Data entry
