Sort the entities

If you have a saves key-list, all entities of one or more collections may be ordered by ab identicative element (or several elements) from each entity,
To sort, on the main menu

click on Sort a result list ()

On the Sorting screen,

you'll have to

  • Enter or find the name of the keylist ()
  • Open the list for each collection, that you want to include in the ordered display ()
  • Check the fields, that compose the sort-key in the correct order. The order is noted with a sequence number after the name, once checked.
  • Click on Sort to order the list.
On the next screen,

Click on Print ()
to go to the well known screen to display the results.

You may write a ReXX program construct the sort-key, if the standard way is not sufficient. See User exits Sorting for details.
