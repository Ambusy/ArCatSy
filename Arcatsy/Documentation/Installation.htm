To install the archive cataloguing system you execute these steps:

  • Unpack all files in the .ZIP file in an (just created) empty folder.

  • Navigate in My Computer to this new folder and execute the set-up program: SetupArCatS.exe

    • If the .ZIP file contains more than one .MDB, you’ll be asked to choose the database to work with. In the Arcives-zip:

      • Choose SAMPLE if you want to practise with (a small collection of) real data taken from the documentation for the original implementation of the Rapide information system at the Katholiek Documentatie Centrum in Nijmegen. Data which you can still consult over the internet. Run SetupAtCatS.exe again to start for real once you know the sample.

      • Or ARCHIVE if you want to start for real. The database contains no data, only definitions. You should start by entering data from your archives.

    • The other .ZIP files contain only one database, so there will be no choice.

    • Vista tries to confuse you about installation errors. Remember that this system is copied, not installed. But let Vista have it's way, it does not harm

    • The system is now operational.

  • In the rest of this documentation you’ll get detailed information on how to customize the system:

    • Add fields in the definitions of the archive

    • Change the print layout of entities

    • Create own programs in Rexx, to incorporate into the system

    • Etc.

  • The system is delivered default in the english language, except for the 3 language specific archives implementations. In the installation folder you'll find 3 files named "system messages XX.txt", where XX reads NL, EN or IT. Substitute the contents of the file "system messages.txt" with one of the "system messages XX.txt" files and the system will speak to you in the othre language. Any change in "system messages.txt" will have a direct effect on the system. Feel free to change any text, that you like better. The only limit is the space on the various screens.

  • The “system messages.txt” contains 3 lines to direct the system:

    • ARCAT210 Update XYZ
      where X Y and Z stand for Y or N:
      X: Y allow updating data, N read-only use of the system
      Y: not used
      Z: Y allow multiple terms while searching, N: allow only 1 term

    • ARCAT208 DSN=Arcatsy
      defines the ODBC databasename

    • ARCAT209 server=127.0.0.1;uid=root;pwd=secret;database=ArCatSy;
      define the MySql access parameters

If you also downloaded the documentation Zip, unpack it in the installation folder, a DOC folder will be added with the documentation in Html format.
