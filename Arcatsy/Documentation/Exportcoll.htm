Export Collection data or definitions

On the main menu

click Maintenance(), and on the maintenance screen
select the collection to export () and click Export collection().

Export Collection definitions

To create a new collection, almost identical to an existing one, you may export the definition of a collection to a file, edit the text if you wish, and then import these definitions into a new collection. Arcatsy then allows to adjust it further.

On the screen that appears,

click Browse () to choose the name of the file to receive the definitions.
check Definitions ().
And finally click Export()

The collection will be exported and you'll return to the maintenance screen.


Export Collection data

To use the data of a collection outsize ArCatSy or to copy data from one pc to another, you may export the data. The format is usable in programs like Excel or Word.

On the screen that appears,

click Browse () to choose the name of the file to receive the data.
check Data ().
Optionally check Export Seqnr () to include the internal sequencenumber of the entries in the exported data. When importing these data in another ArCatSy system, the numbering will be maintained (if absent, the next highest number would be used).
Optionally choose a separator () different from the TAB-symbol, according the the needs of the program, that will import the data.
And finally click Export()

The collection will be exported and you'll return to the maintenance screen.
