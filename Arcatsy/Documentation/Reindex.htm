Re-create the search indices

It is possible to modify the regular expressions or the Rexx program, that govern the process of creating search terms from your entities. If such a modification is made, it is probable that the search index for that collection is no longer valid. So here is the way to actualize the index.


On the main menu

click Maintenance(), and on the mainenance screen

click To be indexed().

In the gird that appears,

enter the collection code () on the empty line followed by a 0 in the CollSeq cell () next to it. After having inserted each collection to be re-processed, click Save ()

When you return to the main menu

select all collections to update () with a (ctrl-)mouse click and click on Update indices ()
and the indices represent the new way of ectracting search terms.
