Import display or print layout

To create a new layout, almost identical to an existing one, you may export the definition of a layout to a file, edit the text if you wish, and then import these definitions into a new layout. Arcatsy then allows to refine it further.
On the main menu

click Maintenance(), and on the maintenance screen

choose Print Layouts () to define the name of the layout that you're about to create. You'll see the grid with the layouts already defined:

In the grid marked select the code of the collection, for which you are about to enter (or modify) a layout ().

In the grid marked click on the empty cell below PrtSpec () and insert a code for the new layout.

Click on Save () to create the layout and then close the screen to return to the maintenance screen.

Select the newly created layout. () and click Import layout().

On the screen that appears,

click Browse () to choose the name of the file that contains the definitions.
And click Import()

The layout will be imported and you'll return to the maintenance screen. You can modify the layout, using the procedure in Define / modify layout
